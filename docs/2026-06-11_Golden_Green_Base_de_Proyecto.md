# Golden Green — Base de Proyecto

Golden Green es un nuevo proyecto de Mao. Ya existe dominio, pero Mao todavía va a explicar exactamente qué es y qué debe construirse.

## Regla de arranque

Golden Green debe diseñarse desde cero usando el **ADN Operativo Mao + Hermes**:

```text
auditar realidad → diseñar → planificar → ejecutar → verificar → documentar
```

## Separación de negocios

Golden Green debe tratarse como proyecto separado de:

- [[TusAguacates]]
- [[BioNeural]]

Solo se conectará con ellos si Mao lo indica explícitamente.

## Intake pendiente

Cuando Mao explique el proyecto, capturar:

- Qué es Golden Green.
- Qué dominio ya compró.
- Qué vende o qué problema resuelve.
- Cliente objetivo.
- País/ciudad/mercado.
- Oferta inicial.
- Modelo de negocio.
- Competidores/referencias.
- Activos existentes: logo, fotos, textos, cuentas, redes, hosting.
- Qué debe tener el MVP.
- Qué cuenta como “listo para probar”.

## Primer entregable esperado

Después de recibir la explicación de Mao, Hermes debe producir:

1. Tesis del proyecto.
2. Propuesta de valor.
3. Mapa de usuario/cliente.
4. MVP recomendado.
5. Arquitectura técnica.
6. Modelo de datos inicial si aplica.
7. Diseño de marca/contenido si aplica.
8. Plan de implementación por fases.
9. Verificación necesaria antes de lanzamiento.
